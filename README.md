# Linuxコマンドが苦手な人へ

## はじめに：

**LPICレベル1の学習支援ツール：「楽しくないものは続かない」** **宝石が降ってくる学習ツール。**という某アニメが好きな人には刺さるプチツールを作りました。

---

## どんなツールか

Claudeと共同で作成したアプリで、こんな機能がある。

- 💎 **25問のLPIC頻出コマンドクイズ**（ファイル操作・パーミッション・ネットワーク・SSH・テキスト処理など）
- 🌸 **正解すると宝石が画面いっぱいに降ってくる**
- ⭐ 難易度 ★〜★★★ のフィルタリング
- 🔀 シャッフル・自動送り機能（6秒〜30秒）
- 📥 **Notion Markdown をそのままインポートできる**（表形式・見出し形式・Q&A形式・番号リスト対応）
- ✏️ 1問ずつ手動追加
- 📊 正答率・戦績ダッシュボード
- 🗂 DB管理画面（セクション別問題数の可視化つき）

コマンドを正解したとき、画面いっぱいにルビーやサファイアが降ってくるので、ちょっと達成感があります！

## 設計のこだわり

### 1. 正解体験を「演出」として設計する

学習ツールの一番の敵は**「正解してもなんも嬉しくない」問題**だ。

```
正解 → "正解！" と表示 → 次の問題
```

人間は**感情的な報酬**がないと行動を繰り返さないそうです。

このツールでは、正解すると：

1. 80個の宝石パーティクルが画面から降ってくる（`@keyframes gem-fall`）
2. 中央に「✨ 完璧よ！」などのランダム賛辞が出現（`@keyframes gem-bloom`）
3. 次の問題へ自動遷移

不正解は揺れるだけ（`@keyframes gem-shake`）。過度に叱責しない設計。

### 2. 宝石テーマ = セクション × カラーの自動マッピング

```javascript
const GEM = {
  ruby:     { crystal: "#c8506a", ... },
  sapphire: { crystal: "#4060c8", ... },
  emerald:  { crystal: "#2a9860", ... },
  amethyst: { crystal: "#7848c0", ... },
  topaz:    { crystal: "#c07820", ... },
  pearl:    { crystal: "#7878b8", ... },
};

// セクションインデックス → 宝石 が自動決定
const sectionGem = s => GEM[GEM_KEYS[SECTIONS.indexOf(s) % GEM_KEYS.length]];
```

「ファイル操作 = ルビー」「ネットワーク = サファイア」のように、**セクションが増えても自動で色が割り当てられる仕様です**

### 3. Notion Markdown 4形式の自動パーサー

Notion でノートを取りながら、そのままコピペでインポートできる。

```
【形式A: テーブル】
| 問題 | 正解 | タグ | セクション | ヒント | 難易度 |
|------|------|------|----------|--------|--------|
| ls -la で何が見える？ | 隠しファイル | ls | ファイル操作 | -a=all | 1 |

【形式B: 見出し】
## セクション名
### 問題タイトル
**Q:** 問題文
**A:** 正解1, 正解2

【形式C: シンプル】
Q: コマンドは？
A: ls -la / ll

【形式D: 番号リスト】
1. 問題文 → 正解
```

パーサーは行を1行ずつ読んで形式を自動判定する。テーブルは `|` の有無、見出し形式は `##` の深さ、Q&A 形式はプレフィックスで分岐する。

### 4. 正解判定のファジーマッチ

```javascript
const judge = (input, answers) => {
  const v = norm(input);
  return answers.some(a => {
    const n = norm(a);
    return v === n || (v.length >= 2 && (n.startsWith(v) || v.startsWith(n)));
  });
};
```

`ls -la` に対して `ls -al` も正解、`chmod u+x test.sh` に対して `chmod +x test.sh` も正解。**実用的に同値なコマンドを複数登録できる。**

## おわりに

LPIC の勉強は「コマンドを暗記する」ことじゃなくて、「コマンドの思想を理解する」ことだと最近思っている。

- `-r` は recursive
- `-h` は human-readable
- `-i` は interactive か identity

オプションの命名規則がわかると、知らないコマンドでも `man` を読む速度が上がる。

このツールがその一歩目になれば嬉しいです。
